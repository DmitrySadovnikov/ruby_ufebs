# RubyUfebs

Обертка для XML форматов УФЕБС 2018 и 2017 годов. Для валидации и сборки XML используются Nokogiri и HappyMapper

## Installation

Добавить в Gemfile:

```ruby
gem 'ruby_ufebs'
```

Затем:

    $ bundle

Или:

    $ gem install ruby_ufebs
